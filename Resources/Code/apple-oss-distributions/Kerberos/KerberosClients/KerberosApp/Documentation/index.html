Kerberos Application Documentation

The Kerberos application on Mac OS X (aka Kerberos.app) is the GUI ticket management application for Kerberos for Macintosh. It allows the user to obtain, destroy, renew and inspect their Kerberos tickets; change their Kerberos password; and change their Kerberos for Macintosh preferences.

In Mac OS X 10.2 and later, the Kerberos management application ships with the OS and is located in /System/Library/CoreServices . The Mac OS X Kerberos Extras can be used to make an alias in a more convenient location.

For Mac OS X 10.1, the Kerberos application is installed as part of Kerberos for Macintosh. You can get the latest release of the Kerberos for Macintosh for Mac OS X 10.1 here.
