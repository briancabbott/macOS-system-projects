dlcompat is a dlopen(3) et.al. compatibility library for Mac OS X/Darwin. To find out about how to load things natively see the NSModule(3) and related man pages, also look at the dlfcn_simple.c file in cvs.

Getting the project from CVS

Use the following commands to get the project from the OpenDarwin CVS server: 

cvs -d :pserver:anonymous@anoncvs.opendarwin.org:/cvs/od login
cvs -d :pserver:anonymous@anoncvs.opendarwin.org:/cvs/od co -d dlcompat proj/dlcompat
When the server asks you for a password you can simply hit return; the password is empty.
