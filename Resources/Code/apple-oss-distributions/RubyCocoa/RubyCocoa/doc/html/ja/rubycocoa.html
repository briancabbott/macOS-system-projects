RubyCocoa

RubyCocoaはMac OS Xアプリケーションを開発するためのMac OS Xフレームワークです。 RubyCocoaにより、RubyでMac OS Xのnativeアプリケーションを開発できます。 また、Ruby拡張ライブラリとして、Mac OS X固有の機能Cocoaを活用したRubyスクリプトを開発できます。

インストール

Mac OS X 10.4(Tiger)上で、システム付属のRuby 1.8.2とともに使用する場合、ダウンロードから最新バージョンのdmgファイルをダウンロードしてインストールするとよいでしょう。その他の場合については、installation.txtをご覧ください。
