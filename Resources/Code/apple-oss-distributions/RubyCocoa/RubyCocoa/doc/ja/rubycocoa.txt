RubyCocoa
=========

[RubyCocoa][1]はMac OS Xアプリケーションを開発するためのMac OS Xフレームワークです。
RubyCocoaにより、[Ruby][10]でMac OS Xのnativeアプリケーションを開発できます。
また、Ruby拡張ライブラリとして、Mac OS X固有の機能[Cocoa][11]を活用したRubyスクリプトを開発できます。


インストール
------------

Mac OS X 10.4(Tiger)上で、システム付属のRuby 1.8.2とともに使用する場合、[ダウンロード][2]から最新バージョンのdmgファイルをダウンロードしてインストールするとよいでしょう。その他の場合については、installation.txtをご覧ください。
