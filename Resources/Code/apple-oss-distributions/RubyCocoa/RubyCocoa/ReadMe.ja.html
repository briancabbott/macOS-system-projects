RubyCocoa - A Ruby/Objective-C Bridge

RubyCocoa は、オブジェクト指向スクリプト言語Rubyを使ってCocoaプログラミングするためのMac OS Xのフレームワークです。Objective-CオブジェクトとRubyオブジェクトとの間で相互にアクセスするためのブリッジとなります。

詳細については、RubyCocoaのWebサイト http://rubycocoa.sourceforge.net/ をご覧ください。また、http://rubycocoa.com/ にもチュートリアルなどの有用な情報があります。

必要な環境

Mac OS X 10.4.8 およびそれ以降

Appleによって配布されているRubyを上書きしている場合、そのRubyは --enable-shared オプション付きでビルドされている必要があります。
