Kerberos for Macintosh 6.0 Release Notes

Kerberos for Macintosh 6.0 is a major new version of Kerberos for Macintosh for Mac OS X. It is only available as part of Mac OS X 10.5 (Leopard).

System Requirements for Kerberos for Macintosh 6.0

  • Mac OS X 10.5 (included with operating system)

New Features and Changes in Kerberos for Macintosh 6.0

  • New Credentials Cache Implementation
  • Deprecation of Kerberos version 4 and KClient APIs
  • 64-bit framework support
  • Based on MIT Kerberos 1.6
  • Numerous bug fixes. See Version History for details.
