Kerberos for Macintosh 5.5 Release Notes

Kerberos for Macintosh 5.5 is a major new version of Kerberos for Macintosh for Mac OS X. It is only available as part of Mac OS X 10.4 (Tiger).

System Requirements for Kerberos for Macintosh 5.5

  • Mac OS X 10.4 (included with operating system)

New Features and Changes in Kerberos for Macintosh 5.5

  • Kerberos Authentication Dialog
    • New Apple-designed icon and appearance
    • Uses a minimized form when presented by an application in the background to avoid stealing focus unexpectedly
  • Kerberos Application
    • Rewritten in Cocoa
    • Improved appearance and performance
    • New Apple-designed icon
    • New realms editor interface for editing the edu.mit.Kerberos file
  • Miscellaneous
    • Includes MIT Kerberos 1.4
    • KLL APIs and command line tools now support file-based ccaches
    • Library support for multithreaded applications
    • Many bug fixes and feature improvements
    • Classic ticket sharing has been deprecated
