GSSAPI in KfM

This documentation only applies to Mac OS X 10.2, 10.3 and 10.4.

Kerberos for Macintosh includes an implementation of the GSSAPI interface to Kerberos v5. GSSAPI is a generic security interface for applications.

Please refer to the papers and documentation describing GSSAPI listed here.

The latest release is included with Mac OS X 10.2 and later.
