CCAPI in KfM

This documentation only applies to Mac OS X 10.2 and later.

The Credentials Cache API is a low level interface for accessing the in-memory credentials cache for Kerberos v4 and Kerberos v5 tickets. The Credentials Cache API is used by other libraries in the Kerberos framework and should not normally be needed by Kerberos applications.
