=head1 NAME

hidd - The HID library userland daemon.

=head1 SYNOPSIS

NONE (launched by launchd)

=head1 DESCRIPTION

As the HID libraries expand, hidd is the place where things better done in userland reside.  
It is not invoked, nor does it have any options.
