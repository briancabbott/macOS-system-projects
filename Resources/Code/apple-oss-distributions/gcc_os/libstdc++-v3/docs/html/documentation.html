The GNU C++ Standard Library

This documentation describes Apple's version of the GNU C++ Standard Library implementation, libstdc++. Information about using libstdc++, including GNU- and Apple-specific extensions, implementation capabilities, or known weaknesses in implementation are described here. This documentation is organized by chapters, where the chapter number references the associated chapter of the ANSI/ISO C++ standard.
