YTFS

YTFS is a a simple read-only file system that downloads a feed of top-rated 
YouTube videos and presents them in the root directory as a clickable webloc 
to the video.

The DEVELOPER_OBJC_HOWTO is a step-by-step tutorial that walks you through 
creating the YTFS file system. Please read and follow the instructions in
the tutorial located here:

http://code.google.com/p/macfuse/wiki/DEVELOPER_OBJC_HOWTO
