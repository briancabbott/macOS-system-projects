ZLGotoSandboxPlugin-Xcode
------------

You can quickly enter the iOS simulator Xcode plugin!
-----
#### Support When running the simulator shortcut keys ('Shift + Common + w') Jump Sandbox. 
#### Run , Restart XCode (common+Q or exit XCode.)
